My first post - Set my own blog

Hexo

Prepare - installation

  1. Node.js
  2. Git
  3. install Hexo
    $ npm install -g hexo-cli

Set up a new blog

$ hexo init blog

$ cd blog

$ npm install

Start a local sever

$ hexo server

now my blog is hot reload at http://localhost:4000/

Deploy my blog to Github

  1. Create a new repository

Repostoryname: ownername.github.io

e.g JinWangQ.github.io

  1. Edit _config.yml
  1. install hexo-deployer-git

$ npm install hexo-deployer-git --save

  1. Upload your site

$ hexo deploy

now http://jinwangq.github.io is available

write a new post (what i am now doing ;D)

  1. Create a new post

$ hexo new "my first post"

  1. Edit source/_posts/My-first-post

Markdown supported
